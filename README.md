# Blood Donor Matcher

A real-time platform that connects patients in need of blood transfusions with nearby donors.

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open `http://localhost:5000` in your browser.
